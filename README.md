## Реализованный функционал

* Экран генерации пользователя с выбором пола и национальности
* Запрос данных к публичному API RandomUser
* Сохранение полученных данных в Room Database
* Отображение списка ранее созданных пользователей
* Экран детальной информации с разбивкой данных по вкладкам
* Удаление пользователя с подтверждением
* Обработка ошибок и отображение информативных сообщений
* Навигации назад согласно требованиям

## Используемые технологии и инструменты
* Kotlin
* Jetpack Compose, Material 3, Coil
* Ktor Client 
* Room
* Hilt
